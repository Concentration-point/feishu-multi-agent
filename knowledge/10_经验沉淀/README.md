# 经验沉淀

这里是正式经验区，不是所有自动生成内容的落点。

## 作用
给 strategist、copywriter、reviewer 以及未来 router/planner 提供跨项目复用经验。

## 正式经验至少应包含
- 场景
- 策略
- 结果
- 教训
- 适用角色
- 来源项目
- 来源运行
- 来源阶段
- 审核状态
- 置信度

## 与收件箱的区别
- `10_经验沉淀`：正式区，要求高
- `11_待整理收件箱`：候选区，允许脏、允许不稳定

## 当前项目接入原则

等后续升格规则明确后，再把合格内容迁入这里。
