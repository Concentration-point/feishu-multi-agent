# 答辩素材包

这目录用于放答辩/PPT可直接复用的材料。

## 已准备内容

- `01-项目概述.md`：一页讲清项目要解决什么问题
- `02-系统架构图.md`：系统组件与数据流
- `03-流水线时序图.md`：五个流水线 Agent 的执行顺序 + 数据分析师独立说明
- `04-记忆系统设计.md`：L0 / L1 / L2 三层记忆
- `05-自进化闭环.md`：经验蒸馏、沉淀、复用
- `06-demo-讲解词.md`：现场演示时可以直接照着讲
- `07-亮点与答辩问答.md`：项目亮点、风险点、常见问答

## 建议用法

1. 先看 `06-demo-讲解词.md`，确保演示口径稳定。
2. PPT 页面优先做成：问题 -> 架构 -> 流程 -> 记忆 -> Demo -> 亮点。
3. 如果时间紧，至少把 02 / 03 / 04 / 06 四份塞进 PPT。
