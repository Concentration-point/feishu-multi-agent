# 小红书爆款选题逻辑

## 选题底层逻辑

小红书用户行为的三种互动动机，决定选题方向：

| 互动行为 | 背后动机 | 对应内容类型 |
|---------|---------|------------|
| 收藏 | 「以后用得上」 | 攻略/清单/教程/对比 |
| 评论 | 「有共鸣」或「有分歧」 | 经验分享/踩坑/反常识 |
| 分享 | 「帮我说出了我说不出的」 | 身份认同/情绪表达/价值观 |

**选题时先问：我希望用户做什么？** 想要高收藏 → 做工具性内容；想要高评论 → 做情绪共鸣或争议性内容；想要高分享 → 做身份认同内容。三种目标不要同时追，一篇笔记服务一个主要互动目标。

## 蓝海词选题法（找竞争少的切入点）

**核心思路**：避开大词的红海竞争，用三维交叉找到搜索量真实存在、但内容供给稀缺的细分选题。

**三维选题模型**：`人群词 × 场景词 × 功效/品类词`

| 维度 | 示例（美妆） | 示例（餐饮） |
|------|------------|------------|
| 人群词 | 孕期妈妈 / 敏感肌 / 学生党 | 一人食 / 带娃家庭 / 外食族 |
| 场景词 | 夏天 / 运动后 / 约会 | 工作日午餐 / 家庭聚会 / 宿舍 |
| 功效/品类词 | 控油 / 防晒 / 粉底 | 快手菜 / 减脂 / 15分钟 |

**示例**：不搜「粉底液」（百万竞品），搜「孕期 可用 粉底」（蓝海，搜索需求真实，内容少）

**蓝海判断标准**（用小红书搜索框验证）：
- 搜索结果 < 500 篇 → 蓝海，值得做
- 搜索结果 500-2000 篇 → 竞争适中，需差异化角度
- 搜索结果 > 2000 篇 → 红海，需要极强的内容质量才能突围

**操作步骤**：
1. 在小红书搜索框输入核心词，观察下拉联想词 → 这些都是真实用户搜索词
2. 点进搜索结果，看前 20 篇的点赞/收藏数 → 平均低但某几篇极高 = 该选题有潜力但现有内容没做好
3. 找数据最好的那篇，分析它的角度，然后做「同选题、更好角度」

## 爆款角度公式

同一个选题，角度决定流量。5 类高出圈角度：

### 1. 反常识揭秘
打破用户既有认知，制造「原来我错了」的冲击感。

- 触发机制：认知冲突 → 用户想知道「为什么」→ 完读率高
- 标题公式：「求求你别再 [常见做法] 了」/ 「[常见做法] 其实是错的」
- 跨行业示例：「洗脸别用温水」（护肤）/ 「冰箱别放这 5 种食物」（家居）

### 2. 踩坑复盘
用真实代价换来的经验，用户信任度高。

- 触发机制：用户想「借鉴别人的失败」来规避风险
- 标题公式：「[金额/时间] 踩坑换来的教训」/ 「后悔没早知道的 X 件事」
- 跨行业示例：「装修花了 30 万踩的 8 个坑」/ 「买了 5 款防晒踩坑报告」

### 3. 极端测试
用极端条件验证产品/方法，满足用户「我没机会亲测但想知道结果」的心理。

- 触发机制：稀缺性（用户没法自己测）+ 客观感
- 标题公式：「[极端条件] 测了 [时长/次数]，结果是」
- 跨行业示例：「游泳 1 小时后防晒还在吗」/ 「同一件衣服洗了 50 次变成这样」

### 4. 小众好物发现
挖掘大众不知道但好用的东西，满足「优越感」和「分享欲」。

- 触发机制：用户想做「给别人安利的人」
- 标题公式：「[行业] 人才知道的 X 个宝藏」/ 「不到 X 元但比大牌好用」
- 跨行业示例：「设计师才知道的 5 个家居品牌」/ 「美容院同款精华，超市就有卖」

### 5. 人群身份代入
用精准人群标签制造「这说的就是我」的代入感，引发收藏和分享。

- 触发机制：身份认同 → 用户转发给朋友「说的就是我」
- 标题公式：「[精准人群] 专属 [解决方案]」/ 「同为 [人群标签]，总结了 X 个方法」
- 跨行业示例：「混油皮的夏天护肤路线」/ 「在上海租房的打工人必备 App」

## 热点借势清单

### 季节 / 节日节点

| 时间段 | 高潜力选题方向 |
|-------|-------------|
| 1-2月（春节） | 年货选购 / 走亲戚穿搭 / 开工仪式感 |
| 3-4月（春季换季） | 换季护肤 / 春季出游 / 春装搭配 |
| 5-6月（毕业季/618） | 毕业礼物 / 618 购物攻略 / 夏日防晒 |
| 7-8月（暑假） | 学生党攻略 / 旅行好物 / 避暑方案 |
| 9-10月（开学/秋季换季） | 开学装备 / 秋季穿搭 / 秋季护肤 |
| 11-12月（双11/年末） | 双11 清单 / 年末总结 / 跨年穿搭 |

**提前量**：节日热点内容建议提前 7-14 天发布，在热点峰值到来时已有互动积累，算法分发效果更好。

### 蹭热点标题结构

- `同样是 [普通行为]，为什么别人 [更好结果]`
- `趁 [热点节日]，整理了 X 个 [实用清单]`
- `[流行词/梗] 版 [品类内容]`（谨慎使用，避免生硬）

### 话题广场借势

发布时选择 **正在涨粉的话题**（不是最大的话题）：
- 进入话题，看「笔记数」× 「最近热度」的比值
- 笔记数少但热度在涨 = 流量洼地
- 话题下已有 10 万+ 笔记 = 竞争太激烈，不如建细分话题
