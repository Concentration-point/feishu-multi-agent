# 抖音脚本模板 — 服饰/穿搭

## 适用客户
服装品牌、穿搭博主、服装店、饰品品牌

## 内容类型选择
| 类型 | 适用场景 | 时长 |
|------|---------|------|
| 穿搭展示 | 单品/全身搭配 | 15-30 秒 |
| 变装卡点 | 风格切换/场景换装 | 15-20 秒 |
| 穿搭教学 | 搭配技巧/显瘦显高 | 30-45 秒 |
| 开箱试穿 | 新品展示/测评 | 30-60 秒 |

## 穿搭展示脚本模板
```
[0-2s] 换装前出镜
画面：家居服/随意状态正面
字幕：「出门前」
BGM：卡点音乐

[3-5s] 换装转场
画面：镜头遮挡/旋转 → 卡点切换

[6-10s] 全身展示
画面：穿搭完成后全身镜像展示
字幕：整套风格定位关键词 + 总价

[11-15s] 上半身细节
画面：上半身 + 领口/袖口/面料质感特写
字幕：上衣名称/价格标注

[16-20s] 下半身 + 配件
画面：下半身走动 + 鞋/包/配饰特写
字幕：下装/配件名称/价格

[21-25s] 走动转化
画面：自然走动展示整套穿搭状态
口播/字幕：「同款链接在购物车」
```

## 穿搭教学脚本模板
```
[0-3s] 钩子
口播：「小个子/微胖姐妹怎么穿显高/显瘦？」
画面：正面出镜

[4-8s] 技巧一说明
画面：正脸出镜，配字幕
口播：「第一，XX（具体原则一句话）」
字幕：技巧名称大字飞入

[9-14s] 技巧一穿搭演示
画面：穿搭展示对应技巧一的效果
口播：「你看这样穿，腰线/腿长/比例就出来了」

[15-19s] 技巧二说明
口播：「第二，XX（技巧二关键点）」
字幕：技巧二关键词

[20-25s] 技巧二穿搭演示
画面：穿搭演示对应技巧二
口播：「对比一下——（前后切换）」

[26-30s] 完整穿搭展示
画面：全身镜展示最终搭配方案
口播：「掌握这几点，XX 身材也能穿出 XX 感」

[31-35s] 转化引导
画面：单品展示 + 购物车方向手势
口播：「想看更多穿搭关注我」
```

## 开箱试穿脚本模板
```
[0-3s] 钩子
画面：快递箱/购物袋出镜
口播：「XX 家的新品到了，帮你们试试值不值得买」

[4-8s] 第一件取出
画面：从箱/袋取出第一件，展开抖开
口播：「第一件，XX 元，先看面料」

[9-13s] 第一件细节
画面：面料拉伸/触感/正反面特写
口播：「手感是 XX，版型是 XX 型，适合 XX 身材」

[14-18s] 第一件上身
画面：穿好后正面+侧面各 2 秒
口播：「上身效果——这个 XX（真实一句话评价）」

[19-23s] 第二件取出+细节
画面：第二件展示，核心细节 2-3 秒特写
口播：「第二件，XX 元，这件主要看 XX」

[24-28s] 第二件上身
画面：穿好后全身展示，可加走动
口播：「版型是 XX，XX 身材直接穿没问题」

[29-34s] 第三件上身（如有）
画面：第三件上身 3-4 秒快速展示
口播：「第三件中规中矩，喜欢 XX 风的可看」

[35-40s] 总结推荐 + 转化
画面：最推荐的那件再次露出
口播：「这次最推荐第 X 件，链接已挂好」
```

## 拍摄要点
- 全身镜/固定机位保证画面稳定
- 面料质感用慢动作展示（甩裙摆、近距特写）
- 光线自然均匀，展示衣服真实颜色
- 变装转场要干净利落，卡点精准

## 发布时间
晚间 19:00-22:00，周二、周四效果较好

## 强钩子库（服饰品类专属）

1. **身材代入**：「我 [身高]cm，体重 [具体/微胖区间]，这套穿上被说像 [正面形容]——我拍下来给你看」
2. **时间验证**：「同事以为我新买的，实际上这件我穿了 [时长]」
3. **价格悬念**：「试了 [数量] 件，只有这一件让我心动，你猜多少钱」
4. **情绪触发**：「微胖姐妹看过来，这条裤子我穿上真的 [情绪反应]，[具体原因]」
5. **季节仪式感**：「[季节] 第一件战袍选好了，没想到是它——[亮点描述]」

**钩子使用原则**：身材数据越具体越能引起代入（写"156cm微胖"比"小个子"共鸣更强）；悬念类钩子要在视频前 15 秒给出答案，否则用户会划走。

## 情绪语言包

**穿搭感受词**（替换"版型好看""显瘦显高"等通用词）：

| 通用描述 | 情绪化替换 |
|---------|-----------|
| 版型很好 | 穿上不想脱那种 / 照镜子照了很久 |
| 显瘦 | 穿上去腰线自动出来了 |
| 显高 | 我妈以为我穿了增高鞋 |
| 面料好 | 摸到面料的时候停了一下，这个价位不应该有这个手感 |
| 很百搭 | 我一周穿了 [次数] 次，每次配的不一样 |
| 性价比高 | [价格] 元，穿出 [倍数] 倍的感觉 |
| 颜色好看 | 这个颜色我拍照的时候图没修就已经很好看了 |

**穿搭口播节奏词**：
「来」「你看」「等等」「重点是这里」「对不对」「这个细节」「我最爱这里」

## 去广告感写法指引

| 硬广写法（❌避免） | 真实感替换（✅用这个） |
|-----------------|-------------------|
| 「采用优质面料，做工精良」 | 「这个缝线我翻过来检查了，比我在实体店买的 [价格更高的品牌] 还整齐」 |
| 「适合多种场合穿着」 | 「我穿着它去上班，周末又穿着它去买菜，都没违和感——这对我来说就是百搭」 |
| 「多色可选，满足需求」 | 「我自己买了 [颜色]，我朋友看到照片直接买了 [另一个颜色]，她说她更适合那个」 |
| 「现在下单立享折扣」 | 「这个价格我截图了，怕之后找不到，你也截一下」 |
| 「版型修身，显瘦效果好」 | 「我平时不敢穿 [款式]，这件我试了一下，拍照给我妈，她说可以穿出去」 |

**核心原则**：穿搭内容最强的真实感来自**尺码/身材数据+具体穿着场景**。"156cm微胖穿了这件去面试"比"显瘦显高显气质"有效 10 倍。
