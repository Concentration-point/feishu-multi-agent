# 抖音脚本模板 — 家居/家装

## 适用客户
家具品牌、装修公司、家居用品、收纳整理、智能家居

## 内容类型选择
| 类型 | 适用场景 | 时长 |
|------|---------|------|
| 好物推荐口播 | 家居单品种草 | 20-40 秒 |
| 空间改造 | 装修案例/收纳前后 | 30-60 秒 |
| 避坑指南 | 装修知识/行业揭秘 | 30-60 秒 |
| 沉浸式家居 | 氛围感/生活方式 | 15-30 秒 |

## 好物推荐脚本模板
```
[0-3s] 钩子
画面：问题场景实拍（凌乱桌面/小户型拥挤）
口播：「小户型收纳头疼？这个东西救了我」

[4-8s] 产品出场
画面：产品完整外观展示/开箱状态
口播：「XX 元，来看看它能做什么」

[9-13s] 安装/使用过程
画面：安装或首次使用的关键步骤特写
口播：「装起来 XX 分钟，不需要打孔」

[14-18s] 效果一（局部对比）
画面：使用后局部效果（整齐/节省空间）
口播：「你看，原来 XX → 现在变成 XX」

[19-23s] 效果二（整体空间）
画面：整个空间改变后的广角
口播：「整体空间感觉多出来了」

[24-30s] 价格 + 转化
画面：产品特写 + 价格字幕
口播：「才 XX 块，链接在购物车，家里乱的赶紧试」
```

## 空间改造脚本模板
```
[0-3s] 改造前全景
画面：原始空间全景（最差状态）
字幕：「花了 XX 元改造出租屋」

[4-8s] 改造前最大问题特写
画面：具体问题区域近景（墙面/堆放/杂乱）
字幕：标注原始状态问题

[9-13s] 改造过程快剪一
画面：大动作工序（刷墙/搬家具）延时快进
BGM：节奏卡点

[14-18s] 改造过程快剪二
画面：布置细节（摆放装饰/铺床品）快进
字幕：标注关键花费

[19-23s] 改造后全景
画面：同机位成果全景（与前形成对照）
口播：「你看，就花了 XX 元」

[24-28s] 最满意区域特写
画面：最满意的区域近距展示
口播：「最满意这个 XX 区域」

[29-35s] 第二亮点 + 转化
画面：第二个亮点区域特写
口播：「整个 XX 元，值不值？评论区说」
```

## 装修避坑脚本模板
```
[0-3s] 钩子
口播：「装修千万别 XX，我花了 XX 万买的教训」
画面：问题现场实拍（漏水/变形/色差）

[4-8s] 坑一描述
画面：坑一的实物近景展示
口播：「第一个坑：XX，当时我以为 XX」
字幕：「坑 ①」大字标注

[9-13s] 坑一解法
画面：正确做法示意/实拍
口播：「正确做法是：XX，能避免 XX」

[14-18s] 坑二描述
画面：坑二的实物展示
口播：「第二个坑，最多人踩：XX」
字幕：「坑 ②」标注

[19-23s] 坑二解法
口播：「应该选 XX，差价不大但结果差很多」

[24-28s] 坑三（最意外的）
画面：坑三实物特写
口播：「第三个是我最没想到的：XX」
字幕：「坑 ③」标注

[29-35s] 总结 + 互动
画面：正脸出镜
口播：「还有装修问题评论区问我，关注我少花冤枉钱」
```

## 拍摄要点
- 空间展示用广角镜头，小空间显大
- 前后对比保持相同机位和角度
- 家居氛围视频注重暖光 + 干净构图
- 收纳类「过程」比「结果」更重要，展示动手过程

## 发布时间
晚间 19:00-21:00，周末效果较好

## 强钩子库（家居品类专属）

1. **低预算改造**：「出租屋住了 [时长]，花 [具体金额] 改造成这样，房东来了都不认识」
2. **行业内幕**：「这个东西，装修公司没告诉我，我 [时间] 之后才发现——[具体是什么]」
3. **对比落差**：「打工人的小窝，乱成这样→收拾成这样，只花了一个周末，[关键做法]」
4. **踩坑升华**：「买了 [年数] 年家居，踩了这几个坑之后，我现在只买 [某类/某种标准] 的」
5. **熟人追问**：「朋友来我家总问同一个问题：[这个在哪买的/这是哪里来的]——今天统一答」

**钩子使用原则**：家居内容的核心情绪是"我也想要这样"和"帮我少走弯路"，钩子要二选一，别两个都要。

## 情绪语言包

**家居体验词**（替换"实用""好看""性价比高"等通用词）：

| 通用描述 | 情绪化替换 |
|---------|-----------|
| 实用 | 买来放了三年，没有一天不在用 |
| 好看 | 放在那里我每次经过都会停一下看 |
| 改变很大 | 住进去第一天早上起来，我在床上发了三分钟呆 |
| 推荐购买 | 我已经买了 [数量] 个送人了 |
| 性价比高 | [价格] 元放了两年，今天检查了一遍，没有任何问题 |
| 空间变大了 | 同样的面积，感觉突然多出来了 |
| 省心 | 从装上去到现在没碰过，它自己就好好的在那里 |

**改造口播节奏词**：
「你看」「对比一下」「重点是」「这里注意」「这一步很关键」「装完之后」「值不值」

## 去广告感写法指引

| 硬广写法（❌避免） | 真实感替换（✅用这个） |
|-----------------|-------------------|
| 「优质材质，经久耐用」 | 「这个我买了 [时长]，搬过 [次数] 次家，现在还在用，每次搬家我都专门把它包起来」 |
| 「安装简便，一学就会」 | 「说明书是英文的，但安装真的不需要看说明书，我一个人 [时长] 装完了」 |
| 「美化家居空间，提升生活品质」 | 「自从放了这个，我朋友来家里第一眼看这里，而不是看电视了」 |
| 「多色可选，百搭风格」 | 「我家是 [风格]，选了 [颜色]，上手之后比我想象的更融合，没有违和感」 |
| 「现有库存有限，建议尽快下单」 | 「这个价格我看到的时候手速快了，你看现在还有没有——[截图/链接]」 |

**核心原则**：家居内容的信任来自**时间验证**（用了多久没坏）和**空间真实感**（实拍不用精修，真实的家比样板间更有说服力）。
