# 抖音脚本模板 — 家居/家装

## 适用客户
家具品牌、装修公司、家居用品、收纳整理、智能家居

## 内容类型选择
| 类型 | 适用场景 | 时长 |
|------|---------|------|
| 好物推荐口播 | 家居单品种草 | 20-40 秒 |
| 空间改造 | 装修案例/收纳前后 | 30-60 秒 |
| 避坑指南 | 装修知识/行业揭秘 | 30-60 秒 |
| 沉浸式家居 | 氛围感/生活方式 | 15-30 秒 |

## 好物推荐脚本模板
```
[0-3s] 钩子（场景痛点）
画面：问题场景（凌乱桌面/小户型拥挤）
口播：「小户型收纳头疼？这个东西救了我」

[3-20s] 产品使用演示
画面：开箱 → 安装/使用 → 效果展示
口播：「它可以 XX，而且 XX，
你看原来这样 → 现在变成这样」

[20-30s] 价格 + 转化
画面：整洁场景 + 产品露出
口播：「才 XX 块，链接在购物车，
家里乱的姐妹赶紧试」
```

## 空间改造脚本模板
```
[0-3s] 改造前全景
画面：原始空间（展示问题）
字幕：「花了 XX 元改造出租屋」

[3-25s] 改造过程（快进 + 卡点）
画面：拆旧 → 刷墙 → 布置 → 装饰（延时/快进）
BGM：节奏卡点音乐
字幕：标注关键步骤和花费

[25-35s] 改造后展示
画面：多角度成果 → 细节特写
口播：「最满意这个 XX 区域，
整个花了 XX 元，值不值？」
```

## 装修避坑脚本模板
```
[0-3s] 钩子
口播：「装修千万别 XX，我花了 XX 万买的教训」

[3-25s] 踩坑经历 + 正确做法
画面：问题实拍 + 口播讲解
口播：讲 1-2 个具体坑 → 每个坑给对应的正确做法

[25-35s] 总结 + 互动
口播：「还有什么装修问题评论区问我，
关注我少花冤枉钱」
```

## 拍摄要点
- 空间展示用广角镜头，小空间显大
- 前后对比保持相同机位和角度
- 家居氛围视频注重暖光 + 干净构图
- 收纳类「过程」比「结果」更重要，展示动手过程
