# 项目档案

这里对齐的是飞书项目主表，是当前系统的项目文档镜像层。

## 作用
把一条飞书项目记录，沉淀成可阅读、可复盘、可被 Agent 再次理解的项目主档案。

## 应与哪些字段对齐
优先对齐项目主表核心字段：
- `client_name`
- `brief`
- `project_type`
- `brand_tone`
- `status`
- `brief_analysis`
- `strategy`
- `review_summary`
- `review_pass_rate`
- `review_red_flag`
- `delivery`
- `knowledge_ref`

## 这层回答什么
- 这个项目是谁的
- 这次要做什么
- 当前做到哪
- 关键策略是什么
- 最终结果是什么

## 和项目执行记录的区别
- 项目档案：项目主线摘要，偏总览
- 项目执行记录：过程材料，偏细节

## 建议命名
`项目档案｜客户名｜项目名`
