# 全案内容营销多智能体系统SOP

## 总览

本文件定义了智策传媒全案内容营销多智能体系统的标准操作流程，涵盖不同项目类型的执行步骤和协作规则。

## 通用流程框架

用户新建 Brief 行
↓
客户经理 → 读 Brief + 品牌调性 → 写 Brief 解读 → 状态=解读中
↓
策略师 → 读 Brief 解读 + 知识空间 → 写策略方案 + 创建内容行 → 状态=策略中
↓
文案 → 逐行读内容排期表 → 逐行写成稿 → 状态=撰写中
↓
审核 → 逐行读成稿 → 逐行写审核状态+反馈 → 状态=审核中
↓ (通过率 < 60% → 驳回到文案返工，最多重试 2 次)
项目经理 → 排发布日期 + 写交付摘要 → 状态=已完成
↓
全程 IM 群聊广播关键节点

## 项目类型专用流程

### 1. 电商大促流程

**适用场景**：双十一、618、年货节等大型电商促销活动

**流程特点**：强调促销信息的准确性、时效性和转化率

用户新建 Brief 行（电商大促）
↓
客户经理 → 读 Brief + 品牌调性 → 写 Brief 解读（重点提取促销力度、时间节点） → 状态=解读中
↓
策略师 → 读 Brief 解读 + 知识空间（参考历史大促方案） → 写策略方案（促销节奏、平台矩阵） + 创建内容行（突出优惠信息） → 状态=策略中
↓
文案 → 逐行读内容排期表 → 逐行写成稿（包含促销信息、有效期、CTA） → 状态=撰写中
↓
审核 → 逐行读成稿（重点审核促销信息准确性、合规性） → 逐行写审核状态+反馈 → 状态=审核中
↓ (通过率 < 60% → 驳回到文案返工，最多重试 2 次)
项目经理 → 排发布日期（按促销节奏） + 写交付摘要（促销效果预期） → 状态=已完成
↓
全程 IM 群聊广播关键节点（促销启动、高潮、收尾）
