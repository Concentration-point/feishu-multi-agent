# 抖音脚本模板 — 服饰/穿搭

## 适用客户
服装品牌、穿搭博主、服装店、饰品品牌

## 内容类型选择
| 类型 | 适用场景 | 时长 |
|------|---------|------|
| 穿搭展示 | 单品/全身搭配 | 15-30 秒 |
| 变装卡点 | 风格切换/场景换装 | 15-20 秒 |
| 穿搭教学 | 搭配技巧/显瘦显高 | 30-45 秒 |
| 开箱试穿 | 新品展示/测评 | 30-60 秒 |

## 穿搭展示脚本模板
```
[0-2s] 日常状态出镜
画面：穿着家居服/随意装扮
字幕：「出门前」
BGM：卡点音乐

[2-5s] 换装转场
画面：镜头遮挡/旋转 → 转场到穿搭完成

[5-20s] 穿搭展示
画面：全身 → 上半身 → 细节（包/鞋/配饰）→ 走动
字幕：标注每件单品名称/价格
口播（可选）：「今天这套 XX 风，
上衣 XX 元，裤子 XX 元，整套不超过 XX」

[20-25s] 转化
口播/字幕：「同款链接在购物车」
```

## 穿搭教学脚本模板
```
[0-3s] 钩子（痛点）
口播：「小个子/微胖姐妹怎么穿显高/显瘦？」
画面：正面出镜

[3-25s] 搭配技巧讲解（2-3 个技巧）
画面：每个技巧配一组穿搭演示
口播：「第一，XX（具体技巧 + 演示）」
字幕：技巧关键词

[25-35s] 完整穿搭展示 + 转化
画面：全身展示 + 走动
口播：「掌握这几点，XX 身材也能穿出 XX 感，
想看更多穿搭关注我」
```

## 开箱试穿脚本模板
```
[0-3s] 钩子
画面：快递箱/购物袋
口播：「XX 家的新品到了，帮你们试试值不值得买」

[3-30s] 逐件试穿（每件 8-10 秒）
画面：拿出衣服 → 展示面料/细节 → 上身效果
口播：每件给一句话评价（面料/版型/适合谁）

[30-40s] 总结推荐 + 转化
口播：「这次最推荐第 X 件，链接已挂好」
```

## 拍摄要点
- 全身镜/固定机位保证画面稳定
- 面料质感用慢动作展示（甩裙摆、近距特写）
- 光线自然均匀，展示衣服真实颜色
- 变装转场要干净利落，卡点精准
