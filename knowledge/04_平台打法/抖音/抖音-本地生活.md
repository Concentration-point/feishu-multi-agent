# 抖音脚本模板 — 本地生活/服务业

## 适用客户
美发店、健身房、汽修店、宠物店、摄影工作室、月子中心、洗车店

## 内容类型选择
| 类型 | 适用场景 | 时长 |
|------|---------|------|
| 服务过程展示 | 技术/手艺展示 | 20-40 秒 |
| 客户变化对比 | 效果背书（发型/健身） | 15-30 秒 |
| 老板/员工 IP | 人设打造/日常 | 20-45 秒 |
| 福利活动 | 团购/到店引流 | 15-25 秒 |

## 服务过程展示脚本模板
```
[0-3s] 钩子（客户问题）
画面：客户到店 + 问题展示
口播：「这位姐姐说她 XX 很久了，今天找我解决」

[3-25s] 服务过程（快进 + 关键步骤实拍）
画面：操作过程 → 关键手法特写 → 中间效果
字幕：简要说明每步在做什么
BGM：轻快/治愈音乐

[25-35s] 成果 + 客户反应
画面：最终效果 → 客户惊喜表情
口播：「看看效果，姐姐自己都不敢认了」

[35-40s] 转化
口播：「想要同款的私信我，地址在主页」
```

## 客户变化对比脚本模板（15-25 秒）
```
[0-2s] 变化前
画面：客户原始状态（正面/侧面）
字幕：「改造前」
BGM：卡点音乐

[2-5s] 服务过程快闪
画面：3-5 个操作镜头快速切换

[5-20s] 变化后
画面：成果多角度展示 → 客户满意反应
字幕：「改造后」+ 服务项目名称

[20-25s] 转化
字幕：「同城到店体验，私信预约」
```

## 福利活动脚本模板
```
[0-3s] 钩子（利益先行）
口播：「XX 区的朋友有福了，XX 只要 XX 元」

[3-15s] 展示服务/产品
画面：服务过程/产品实拍
口播：简要说明包含什么内容

[15-22s] 限时限量 + 转化
口播：「只限 XX 份，团购链接已经挂好，
手慢无，赶紧抢」
```

## 拍摄要点
- 重点是「变化感」，前后对比要明显
- 操作手法用近景/特写体现专业度
- 客户反应真实自然，不要摆拍感
- 可用定机位延时拍摄整个服务过程
