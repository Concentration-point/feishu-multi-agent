# 平台内容规范

本文档供审核 Agent 判断内容是否适配目标平台。

## 小红书平台规范

### 内容长度

- 种草笔记建议 300-800 字
- 标题建议 12-28 字
- 开头 3 秒必须抓住用户注意力
- 正文建议分段清晰，避免长段堆叠

### 内容风格

小红书内容适合：

- 真实体验
- 个人故事
- 前后对比
- 使用场景
- 情绪共鸣
- 清单式表达

高风险表达：

- 明显广告腔
- 过度夸大功效
- 无依据数据
- 虚构用户体验
- 诱导交易过强

### 标签建议

- 标签数量建议 3-8 个
- 标签应与品类、场景、人群相关
- 不建议堆砌无关热词

## 抖音平台规范

### 脚本长度

- 短视频脚本建议 30-90 秒
- 前 3 秒必须有钩子
- 每 5-8 秒应有信息点或视觉转折

### 脚本结构

推荐结构：

1. 痛点钩子
2. 场景代入
3. 产品卖点
4. 使用方式
5. 行动号召

高风险表达：

- 夸大承诺
- 口播中使用绝对化用语
- 无依据功效对比
- 低俗博眼球表达

## 公众号平台规范

### 内容长度

- 深度文建议 1200-2500 字
- 标题建议 15-32 字
- 需要有清晰小标题和逻辑层次

### 内容风格

适合：

- 观点阐述
- 品牌理念
- 深度测评
- 成分或技术解析
- 用户教育

高风险表达：

- 标题党
- 数据无来源
- 事实判断无依据
- 过度营销导致阅读体验差

## 微博平台规范

- 文案需要短平快
- 适合话题传播和事件借势
- 避免复杂长逻辑
- 话题标签不宜过多
- 不得蹭不相关热点

## 审核要求

审核 Agent 需要根据内容的目标平台判断：

1. 字数是否大体适配
2. 结构是否符合平台阅读习惯
3. 表达是否过度广告化
4. 是否存在平台高风险内容
5. 是否需要调整标题、开头或 CTA
