# 广告法禁用词清单（全平台通用）

> 本文件供审核 Agent 做合规检查，文案中命中以下任一类别即判「需修改」。
> 违反广告法罚款 20 万起步，最高 100 万。

## 一、绝对化极限用语（一票否决）

### 「最」系列
最佳、最好、最大、最小、最高、最低、最优、最强、最便宜、最先进、最流行、最受欢迎、最时尚、最舒适、最科学、史上最低价

### 「第一/唯一」系列
第一、NO.1、Top1、唯一、独一无二、首个、首选、首家、首发、首款、全网第一、销量第一、排名第一、全国第一、中国第一

### 「顶级/极致」系列
顶级、极致、极品、顶尖、终极、至尊、巅峰、王者、冠军、王牌、金牌、领袖、掌门人

### 「绝对」系列
绝对、100%、万能、永久、一流、绝无仅有、史无前例、前无古人、无人能及

### 审核判定规则
- 直接出现以上词汇 → 判「需修改」
- 变体写法（拼音缩写、谐音、符号分隔如「最·好」）→ 同样判「需修改」
- **例外**：引用客观排名数据并标注来源的（如「2024 年天猫品类销量 Top3，数据来源：XX 报告」）可通过

## 二、虚假承诺用语

### 效果承诺
无效退款、保证有效、立竿见影、X 天见效、一次见效、永不反弹、根治、治愈、药到病除、零风险

### 权威背书造假
国家级产品、国家免检、质量免检、国家领导人推荐、国家机关推荐/专供/特供、驰名商标（未经认定不得使用）、填补国内空白、专家推荐（无具体专家信息）

### 时间压迫（无具体时间）
随时结束、仅此一次、再不抢就没了、不会再便宜了、错过就没机会了、马上降价、随时涨价、最后一波（未标明截止日期）

### 审核判定规则
- 承诺性词汇 + 无数据/无来源 → 判「需修改」
- 限时促销必须标明具体活动起止日期，否则 → 判「需修改」

## 三、功效类禁用词（普通商品不得使用医疗用语）

### 疾病治疗类
治疗、治愈、消炎、抗炎、杀菌、灭菌、消毒、排毒、抗癌、防癌、降血压、降血糖

### 身体机能调节类
增强/提高免疫力、调节内分泌、促进新陈代谢、滋阴补阳、壮阳、助眠、延年益寿、活血、补血、安神、养脑、通脉

### 美妆护肤禁用功效词
减肥、瘦身、瘦脸、丰胸、溶脂、燃烧脂肪、祛疤、除皱、平皱、一洗白、速白、脱敏、抗敏、再生细胞、修复断裂弹性纤维、生发、止脱、永不褪色

### 审核判定规则
- 普通商品（非药品/医疗器械/特殊化妆品）使用以上词汇 → 判「需修改」
- 即使口语化表达（如「喝了它我免疫力都变好了」）也属违规
- **例外**：特殊用途化妆品（如防晒 SPF 标注）、保健食品（有蓝帽子标志）在批准范围内可使用

## 四、诱导/欺诈类

点击有惊喜、恭喜获奖、全民免单、点击领取、点击翻转、秒杀、万人疯抢、抢疯了、卖疯了、错过不再、限量（未标明具体数量）

### 审核判定规则
- 出现以上诱导用语 → 判「需修改」

## 五、歧视/迷信/暴力类（一票否决）

### 歧视用语
任何民族、种族、性别、宗教歧视用语

### 迷信用语
招财进宝、逢凶化吉、转运、旺财、化解小人、增强第六感、算命、算卦等

### 暴力恐怖
杀戮、宣扬武力、恐怖威胁类表述

### 审核判定规则
- 命中 → 直接判「驳回」，不给修改机会

## 六、安全替换建议

| 违规词 | 安全替代 |
|--------|---------|
| 最好 | 很不错 / 个人觉得很好 |
| 第一 | 领先 / 头部 / 知名 |
| 100% | 大部分 / 绝大多数 |
| 永久 | 持久 / 长效 |
| 最便宜 | 性价比高 / 价格友好 |
| 无效退款 | 支持售后 |
| 立竿见影 | 体验感不错 |
| 祛痘 | 帮助净化肌肤 |
| 美白 | 提亮肤色 |
| 减肥 | 轻体 / 体态管理 |
