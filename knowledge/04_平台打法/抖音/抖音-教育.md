# 抖音脚本模板 — 教育/知识/培训

## 适用客户
培训机构、在线教育、知识博主、职业技能、考证辅导

## 内容类型选择
| 类型 | 适用场景 | 时长 |
|------|---------|------|
| 知识点口播 | 单一知识分享 | 30-60 秒 |
| 行业揭秘 | 信息差输出 | 30-60 秒 |
| 学员案例 | 效果背书 | 20-40 秒 |
| 干货清单 | 工具/方法论分享 | 30-45 秒 |

## 知识口播脚本模板
```
[0-3s] 钩子（颠覆认知/抛出问题）
画面：正脸出镜，表情认真
口播：「为什么你背了 100 个单词第二天全忘了？」
      或「XX 行业 90% 的人都踩过这个坑」

[3-30s] 知识输出（1 个核心观点 + 1-2 个论据）
画面：口播 + 关键词字幕飞入
口播：是什么 → 为什么 → 怎么做
字幕：关键数字/关键词放大加粗

[30-40s] 总结 + 互动
画面：正脸出镜
口播：「学会了吗？想知道 XX 的扣 1」
      或「关注我，每天教你一个 XX 技巧」
```

## 行业揭秘脚本模板
```
[0-3s] 钩子
口播：「做了 X 年 XX，今天告诉你一个真相」

[3-35s] 揭秘内容
口播：行业通病/潜规则 → 
你踩坑不是因为你不聪明，是因为 XX →
正确的做法是 XX

[35-45s] 转化
口播：「知道了吧？关注我不踩坑，
想学 XX 的评论区留言」
```

## 学员案例脚本模板
```
[0-3s] 钩子（结果先行）
画面：学员成果展示（证书/成绩/作品）
口播：「从零基础到 XX，她只用了 X 个月」

[3-25s] 学习过程
画面：学员学习场景 / 采访学员
口播：之前什么状态 → 遇到什么困难 →
怎么学的 → 最终结果

[25-35s] 转化
口播：「想要同款学习方案的私信我」
```

## 拍摄要点
- 背景干净（纯色墙/书架/办公室）
- 语速快但清晰，剪掉所有气口
- 重点内容用字幕强化，关键词放大/变色
- 可用手势/道具/白板辅助讲解
- 人设感 > 知识密度
