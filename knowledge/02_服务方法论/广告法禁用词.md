# 广告法与内容合规禁用词

本文档供审核 Agent 在审核营销内容时作为客观规则依据。

## 一、绝对化用语禁用

营销内容不得使用容易构成绝对化承诺或最高级宣传的词语，包括但不限于：

- 最
- 最佳
- 最强
- 最优
- 最好
- 第一
- 全国第一
- 全网第一
- 行业第一
- 顶级
- 极致
- 绝对
- 绝无仅有
- 独家
- 唯一
- 首选
- 必选
- 必买
- 王牌
- 终极
- 巅峰
- 神级
- 天花板
- 完美
- 100% 有效
- 永久有效
- 彻底解决
- 一次见效
- 立刻见效
- 保证有效
- 无副作用

## 二、化妆品与护肤品禁用功效表达

化妆品、护肤品内容不得使用医疗化、治疗化表达，包括但不限于：

- 治疗
- 根治
- 治愈
- 消除炎症
- 抗菌
- 杀菌
- 消炎
- 药用
- 医疗级
- 医美替代
- 修复疾病
- 改变肤质
- 逆转衰老
- 去除皱纹
- 祛除色斑
- 保证不过敏
- 敏感肌百分百适用

允许更稳妥的表达：

- 改善肌肤干燥状态
- 帮助维持肌肤水润
- 提升使用后的滋润感
- 适合多数干燥肌肤场景
- 使用体验更温和
- 有助于修护肌肤屏障感受

## 三、保健品与健康类内容禁用表达

保健品和健康类内容不得暗示疾病预防、治疗、诊断效果，包括但不限于：

- 预防疾病
- 治愈疾病
- 诊断
- 替代药物
- 降血压
- 降血糖
- 消除病症
- 医生推荐必吃
- 治疗失眠
- 改善疾病

## 四、竞品比较风险

不得使用缺乏证据的贬低性比较，例如：

- 比某品牌更有效
- 秒杀同行
- 吊打竞品
- 其他产品都不行
- 市面上只有我们能做到

可替代表达：

- 在同类产品中具备差异化特点
- 更适合注重温和体验的人群
- 采用了不同于传统配方的设计思路

## 五、替代表达建议

| 高风险表达 | 建议替换 |
| --- | --- |
| 最有效 | 表现较优 / 体验突出 |
| 顶级配方 | 高标准配方 / 精选配方 |
| 绝对安全 | 更注重温和性与安全性 |
| 彻底解决 | 帮助改善 |
| 唯一选择 | 值得考虑的选择 |
| 永久有效 | 长期坚持使用更能体现效果 |

## 六、审核要求

审核 Agent 发现上述表达时，应在审核反馈中明确写出：

1. 违规词或风险表达
2. 出现位置
3. 违反的规则文件：广告法禁用词.md
4. 建议替换表达
