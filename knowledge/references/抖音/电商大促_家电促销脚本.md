# 【30秒抖音口播脚本】双十一扫地机器人

**时长：30 秒**
**镜头语言：快切 + 画中画对比**

---

**00:00-00:03【强钩子】**
（镜头：主播面对面正脸）
**台词**：姐妹，你家扫地机是不是用一个月就开始「假扫」？

**00:03-00:08【场景痛点】**
（镜头切：脏乱的地板，头发 + 宠物毛 + 灰尘特写）
**台词**：毛发缠轮子，边角扫不到，拖完地板还黏。我家猫毛我试过三台都败下阵来。

**00:08-00:15【功能演示】**
（镜头切：新机工作画面，快切三个关键功能）
**台词**：
- 这个 **剃须刀级滚刷**，毛发不缠绕，我测了一个月没清过
- **激光 SLAM 建图**，床底沙发底全能钻
- **65℃热水洗拖布**，油污一擦就掉

**00:15-00:22【对比+种草】**
（镜头切：画中画，左旧机打结右新机光滑）
**台词**：这是我上一台 3000 块买的，这是今年新款。差距肉眼可见，关键价格还便宜了一半。

**00:22-00:27【价格揭晓】**
（镜头切：主播手持产品，价格字卡飞入）
**台词**：双十一到手价 **1999**！比去年旗舰便宜整整 2000，而且 **30 天价保**！

**00:27-00:30【紧迫感+CTA】**
（镜头切：倒计时 + 红色字卡）
**台词**：限量 2000 台，点小黄车直接下单，拍完发我截图送一年耗材！

---

## 脚本节奏要点

1. **前 3 秒**必须有钩子（问题/反常识观点），决定 80% 完播
2. **痛点场景**必须具体到"毛发 + 猫毛 + 黏腻"三个细节，避免模糊
3. **功能点**不超过 3 个，每个 2 秒镜头，不给观众思考空间
4. **价格揭晓**放在 22 秒，此时已经完成种草，价格是临门一脚
5. **CTA 必须有紧迫感**：限量 + 赠品 + 价保 三重驱动

## 文案通用公式

> 钩子疑问 → 痛点共鸣 → 产品演示 → 对比反差 → 价格刺激 → 限量 CTA
