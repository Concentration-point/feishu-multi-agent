# 抖音脚本模板 — 美妆/护肤

## 适用客户
护肤品牌、彩妆品牌、美容院、医美机构、个护品牌

## 内容类型选择
| 类型 | 适用场景 | 时长 |
|------|---------|------|
| 产品测评口播 | 新品种草/成分解析 | 45-90 秒 |
| 妆效对比 | 彩妆推广（before/after） | 15-30 秒 |
| 教程演示 | 化妆教学/护肤步骤 | 30-60 秒 |
| 成分科普 | 专业人设打造 | 30-60 秒 |
| 剧情植入 | 品牌软性种草 | 30-60 秒 |

## 产品测评口播脚本模板
```
[0-3s] 钩子
画面：怼脸皮肤问题特写 / 手持产品出镜
口播：「油皮姐妹还在为脱妆烦恼吗？」
      或「这款 XX 到底值不值得买？」

[4-8s] 产品出场
画面：手持产品正面，展示包装与规格
口播：「我用了 XX 之后，有点不敢说实话」

[9-13s] 取量展示
画面：开盖/挤出动作特写 → 手背展示质地
口播：「质地是 XX 的，主打成分 XX」

[14-18s] 上肤涂抹
画面：产品点按/涂抹上脸手部特写
口播：「针对的就是 XX 问题，上脸感受是 XX」

[19-23s] 即时肤感
画面：涂抹后皮肤近景（光泽/服帖/遮盖度）
口播：「你看，上脸之后 XX（具体变化）」

[24-28s] 时效验证
画面：经若干小时后皮肤状态，或计时字幕辅助
口播：「XX 小时之后还是这个状态——」

[29-33s] 前后对比
画面：分屏或翻拍同机位 before/after
口播：「前后放一下，自己看」

[34-38s] 总结推荐
画面：正脸出镜，表情自然
口播：「XX 肤质的姐妹直接冲」

[39-44s] 转化引导
画面：产品正面再次露出
口播：「链接左下角，我自己回购清单里的」
```

## 妆效对比脚本模板（25 秒）
```
[0-2s] 素颜出镜
画面：素颜怼脸特写
字幕：「化妆前」
BGM：卡点音乐

[3-5s] 遮脸转场
画面：用手/刷子遮脸 → 卡点切换

[6-10s] 完妆正脸出场
画面：完妆正脸，慢慢移近镜头
字幕：标注核心产品名

[11-15s] 细节特写一（眼/唇/底妆 选其一）
画面：局部妆效近距特写
字幕：产品名称 + 色号

[16-20s] 细节特写二（另一部位或侧脸）
画面：另一角度/部位特写
字幕：标注使用产品

[21-25s] 转化
画面：整体展示 + 手势引导购物车方向
字幕/口播：「同款链接在购物车」
```

## 成分科普脚本模板
```
[0-3s] 钩子
口播：「为什么你用了 XX 还是没效果？」
画面：正脸出镜，表情认真

[4-7s] 答案点题
口播：「因为你根本没搞懂 XX 成分」
字幕：成分名称大字飞入

[8-12s] 成分是什么
画面：口播 + 字幕辅助
口播：「这个成分简单说就是 XX，它的作用是 XX」

[13-17s] 为什么有效
口播：「为什么有效？因为 XX（底层机制）」
字幕：关键作用词放大

[18-22s] 怎么看配方
画面：（可选）产品背标成分表特写
口播：「挑产品时看成分表，XX 排前三才够量」

[23-27s] 什么肤质适合
口播：「XX 肤质直接用，XX 肤质注意 XX」
字幕：适合 / 不适合 分栏标注

[28-33s] 推荐产品一
画面：第一款产品展示
口播：「这款 XX 成分浓度 XX，我自己最推」

[34-40s] 推荐产品二 + 转化
画面：两款产品并排
口播：「这款适合 XX，链接放好了」
```

## 拍摄要点
- 面部打柔光（环形灯/柔光箱），避免硬阴影
- 产品特写用微距展示质地（膏体/液态/粉质）
- 上脸试用给真实肤感特写，不靠滤镜
- 前后对比保持固定机位 + 统一光线
- 强对比产品（口红/粉底/遮瑕）适合短平快妆效展示

## 发布时间
晚间 19:00-22:00，周二、周四效果较好

## 强钩子库（美妆品类专属）

以下 5 条直接可用，替换 `[具体信息]` 后填入脚本开头：

1. **痛点精准打击**：「油皮姐妹注意了，你以为控油失败是产品的问题——不是，是你忽略了 [具体步骤]」
2. **后悔式警示**：「买这个色号之前千万先看一下，我后悔了三个月，[具体原因]」
3. **认知颠覆**：「护肤品用了 [时长]，今天才知道我一直用错了顺序，[具体说哪个步骤]」
4. **问题归因转移**：「你的粉底不服帖？不是粉底差，是你少做了 [这一步]，照着试试」
5. **时间锚点种草**：「[年龄] 之前用这个，[年龄] 之后你会谢谢自己——[产品名/成分名]」

**钩子使用原则**：数字越具体越好（"三个月"比"很久"强），身份标签要精准（"油皮姐妹"比"姐妹们"强）。

## 情绪语言包

**高频情绪词**（替换掉"效果不错""值得购买"等书面词）：

| 书面感写法 | 情绪化替换 |
|-----------|-----------|
| 效果很好 | 真的救了我 / 绝了 / 用完不敢相信 |
| 值得购买 | 直接冲 / 闭眼买 / 不用犹豫 |
| 香味很好 | 香得我站在洗手间不想出来 |
| 肤感轻盈 | 上脸感觉什么都没有，但摸起来软 |
| 显色度高 | 涂了一秒脸就亮了那种 |
| 已经回购 | 买到第 X 瓶了，每次快用完就焦虑 |
| 推荐购买 | 我已经安利了我妈我姐我闺蜜 |
| 很喜欢 | 破防了 / 上头了 / 爱了爱了 |

**口播节奏词**（增加说话感，放在句子转折处）：
「你看」「然后啊」「真的」「说真的」「不骗你」「就是那种感觉」「懂吧」

## 去广告感写法指引

**识别硬广信号 → 替换为真实感表达**：

| 硬广写法（❌避免） | 真实感替换（✅用这个） |
|-----------------|-------------------|
| 「产品采用 XX 技术，成分安全温和」 | 「我敏感肌，用了三天没红没刺激，这对我来说已经很难得了」 |
| 「适合所有肤质，老少皆宜」 | 「我干皮，我闺蜜油皮，我们两个都在用同一款，她说控油我说保湿，同一瓶各取所需」 |
| 「效果显著，坚持使用」 | 「第七天早上对镜子愣了一下，不是变白了，是皮肤状态整个不一样了」 |
| 「高性价比，物超所值」 | 「XX 元，够用三个月，平摊下来一天不到两块，我没找到不买的理由」 |
| 「现在下单立享优惠」 | 「趁现在，手慢真的会断货，我上次拖了一周结果等了一个月」 |

**核心原则**：承认一个真实的小缺点，反而比"完美"更可信。例：「包装有点朴素，但我不管这个，我管的是用完之后脸的感受」。
