# 抖音脚本模板 — 美妆/护肤

## 适用客户
护肤品牌、彩妆品牌、美容院、医美机构、个护品牌

## 内容类型选择
| 类型 | 适用场景 | 时长 |
|------|---------|------|
| 产品测评口播 | 新品种草/成分解析 | 45-90 秒 |
| 妆效对比 | 彩妆推广（before/after） | 15-30 秒 |
| 教程演示 | 化妆教学/护肤步骤 | 30-60 秒 |
| 成分科普 | 专业人设打造 | 30-60 秒 |
| 剧情植入 | 品牌软性种草 | 30-60 秒 |

## 产品测评口播脚本模板
```
[0-3s] 钩子（痛点切入）
画面：怼脸拍皮肤问题 / 手持产品
口播：「油皮姐妹还在为脱妆烦恼吗？」
      或「这款 XX 到底值不值得买？」

[3-15s] 产品展示 + 核心卖点（最多 2 个）
画面：产品多角度 → 取用质地特写 → 上手/上脸
口播：「质地是 XX，主打成分 XX，
针对的就是 XX 问题」

[15-35s] 使用效果 + 真实感受
画面：使用过程 → 前后对比（分屏或同框）
口播：「上脸之后你看，XX（具体变化），
XX 小时后还是这个状态」

[35-45s] 总结 + 转化
画面：正脸出镜 + 产品露出
口播：「XX 肤质的姐妹直接冲，链接左下角」
```

## 妆效对比脚本模板（15-25 秒）
```
[0-2s] 素颜出镜
画面：素颜怼脸 + 文字「化妆前」
BGM：卡点音乐

[2-5s] 快速变妆转场
画面：用手/刷子遮脸 → 转场

[5-20s] 完妆展示
画面：不同角度展示妆效 + 产品闪过
字幕：标注使用产品名

[20-25s] 转化
字幕/口播：「同款链接在购物车」
```

## 成分科普脚本模板
```
[0-3s] 钩子
口播：「为什么你用了 XX 还是没效果？
因为你根本没搞懂 XX 成分」

[3-30s] 科普输出
画面：口播 + 成分关键词字幕飞入
口播：这个成分是什么 → 为什么有效 → 
怎么看浓度/配方 → 什么肤质适合

[30-40s] 产品推荐 + 转化
画面：推荐 1-2 款含该成分的产品
口播：「想要 XX 效果的，认准这个成分，
这两款我自己用下来最推荐」
```

## 拍摄要点
- 面部打柔光（环形灯/柔光箱），避免硬阴影
- 产品特写用微距展示质地（膏体/液态/粉质）
- 上脸试用给真实肤感特写，不靠滤镜
- 前后对比保持固定机位 + 统一光线
- 强对比产品（口红/粉底/遮瑕）适合短平快妆效展示
