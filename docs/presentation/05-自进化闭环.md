# 自进化闭环

## 闭环流程

1. Agent 完成任务
2. Hook 回看执行过程
3. 蒸馏一张 SAOL 经验卡片
4. 计算置信度
5. 达标后写入经验池 + 本地 Wiki
6. 下次同类项目启动时注入经验

## Mermaid 图

```mermaid
flowchart LR
    A[Agent 执行完成] --> B[Hook Reflect]
    B --> C[生成经验卡片]
    C --> D[置信度打分]
    D -->|达标| E[写入 Bitable 经验池]
    D -->|达标| F[写入本地 Wiki]
    F --> G[同步到飞书知识空间]
    E --> H[下一次任务 get_experience]
    F --> I[下一次任务 search_knowledge]
```

## 讲解重点

- 项目不是静态 Agent 编排，而是**带学习能力的工作流**
- 经验沉淀不是单纯日志，而是经过结构化提炼与置信度筛选
