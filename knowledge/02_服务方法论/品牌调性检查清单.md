# 品牌调性检查清单

本文档供审核 Agent 判断内容是否符合品牌调性。

## 科技类品牌

### 推荐表达

- 使用数据、机制、成分、技术路径说明价值
- 语气克制、专业、清晰
- 少用感叹号
- 少用情绪化夸张词
- 强调逻辑和证据

### 禁用或慎用表达

- 太神了
- 秒变
- 绝绝子
- 神仙产品
- 一用就爱上
- 救命神器
- 无脑冲

### 审核重点

- 是否有清楚的技术或机制解释
- 是否避免过度情绪化
- 是否存在无法证明的效果承诺

## 母婴类品牌

### 推荐表达

- 安心
- 温和
- 陪伴
- 科学喂养
- 安全感
- 真实育儿场景

### 禁用或慎用表达

- 保证宝宝不生病
- 彻底解决喂养问题
- 所有宝宝都适用
- 医生强制推荐
- 绝对安全

### 审核重点

- 是否避免绝对承诺
- 是否突出安全和科学依据
- 是否尊重家长真实焦虑，不贩卖恐慌

## 美妆护肤类品牌

### 推荐表达

- 使用体验
- 肌肤感受
- 成分特点
- 适用场景
- 温和、保湿、修护感

### 禁用或慎用表达

- 治疗皮肤问题
- 根治敏感
- 一夜变白
- 永久去皱
- 医美平替
- 秒杀大牌

### 审核重点

- 是否医疗化表述
- 是否夸大功效
- 是否与目标人群和使用场景匹配

## 新消费品牌

### 推荐表达

- 审美
- 生活方式
- 人群态度
- 场景价值
- 品牌主张

### 禁用或慎用表达

- 空泛口号
- 无依据高端感
- 虚假稀缺
- 盲目制造身份焦虑

## 审核要求

审核 Agent 需要在反馈中说明：

1. 内容当前调性是否匹配
2. 哪些表达偏离品牌调性
3. 应该换成什么表达风格
4. 是否影响发布判断
