## 品牌定位

国民经典运动休闲鞋品牌。以「经典、时尚、专业」为品牌内涵主题，坚持「以人为本、崇尚运动、促进健康」的产品理念。核心定位是**高性价比的大众国货运动休闲鞋**，兼顾经典复古与新国潮时尚。

## brand_tone（品牌调性）

```
国货经典 | 平价亲民 | 复古怀旧 | 年轻化国潮 | 接地气
```

详细说明：
- **国货自信**：百年老字号，强调民族品牌传承和文化底蕴，但不自吹自擂
- **平价亲民**：主力价格带 50-200 元，性价比是核心卖点，表达上要「真实不装」
- **复古怀旧**：经典款（WB-1 篮球鞋、小白鞋）是品牌符号，内容可打情怀牌但不能过度消费
- **年轻化国潮**：近年积极做联名、跨界，走新国潮路线，内容风格可以活泼、潮流
- **接地气**：用户以学生党、年轻上班族为主，文案风格要说人话，拒绝端着

## 常见审核偏好

### ✅ 鼓励的内容方向
- 经典款穿搭（百搭、学生党友好、通勤搭配）
- 性价比突出（「不到 XX 元」「平替」「学生党闭眼入」）
- 国货情怀（品牌历史、民族品牌自信、老照片对比）
- 联名/跨界款展示（非遗、高校合作等正面联名）
- 真实用户体验（穿着感受、耐穿度、实拍）

### ⚠️ 需谨慎的内容方向
- 与国际品牌的直接对比（容易触发「抄袭」争议联想）
- 「最便宜」「第一」等绝对化用语（广告法违规）
- 过度打情怀牌（容易被吐槽「卖惨营销」「消费情怀」）
- 强调「潮牌」定位（回力目前用户心智更偏「平价国货」而非「潮牌」）

## 风险禁区（红线）

| 风险项 | 说明 | 内容处理 |
|--------|------|---------|
| 抄袭争议 | 回力部分鞋款曾被指借鉴 Nike/Converse/Vans 等品牌设计，是品牌最大舆论风险 | **绝对不要**在内容中提及与其他品牌的外观相似性，不做「平替 XX」的对标表述 |
| 品牌授权乱象 | 回力采用轻资产品牌授权模式，经销商自主设计生产，品控参差 | 内容中不要承诺品质（如「品质保障」「绝对正品」），不评论授权体系 |
| 质量事件 | 童鞋曾被检出有害成分（邻苯二甲酸酯），多次上质量黑榜 | **绝对不要**在内容中做「安全无毒」「品质检测合格」等承诺性表述 |
| 与飞跃的关系 | 回力与飞跃有复杂的历史和商标纠纷 | 不要在内容中将两个品牌混为一谈或做对比 |
| 价格战话术 | 回力价格已经很低，再用「跳楼价」「亏本卖」等话术会拉低品牌形象 | 用「性价比高」「价格友好」替代，不做价格战表达 |
