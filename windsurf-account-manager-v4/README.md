# Windsurf Account Manager v4

这版不是“记账网页”，而是按账号管理器做的本地工具：

- 账号池表格
- 当前账号
- 独立浏览器 Profile 登录
- 每个邮箱一个浏览器会话
- 额度 chip / 进度条
- 到期状态
- 批量导入邮箱
- 官方页面文本解析额度
- 导入 / 导出备份

## 使用

1. 解压 zip
2. 双击 `launch-desktop.bat`
3. 添加账号，只填邮箱/昵称即可
4. 点「登录/额度」
5. 第一次在打开的浏览器里自己登录 Windsurf
6. 以后同一个邮箱会复用这个独立浏览器 Profile
7. 在官方用量页复制文本，回工具「编辑」里粘贴并解析额度

## 边界

- 不自动注册
- 不自动开试用
- 不补 token
- 不改 Windsurf 本地登录态
- 不绕平台额度
