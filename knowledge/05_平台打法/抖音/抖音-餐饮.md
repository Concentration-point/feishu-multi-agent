# 抖音脚本模板 — 餐饮/美食

## 适用客户
餐厅、奶茶店、烘焙店、火锅店、地方特色小吃、食品品牌

## 内容类型选择
| 类型 | 适用场景 | 时长 |
|------|---------|------|
| 探店口播 | 新店开业/团购推广 | 30-45 秒 |
| 菜品特写 | 招牌菜/新品上市 | 15-20 秒 |
| 后厨揭秘 | 品质背书/食材展示 | 20-40 秒 |
| 吃播反应 | 口味种草 | 15-30 秒 |
| 老板 IP | 品牌故事/行业揭秘 | 30-60 秒 |

## 探店口播脚本模板
```
[0-3s] 钩子
画面：推门进店瞬间 / 菜品第一眼大特写
口播：「在 XX 路发现一家 XX，人均才 XX 元！」

[4-8s] 环境扫视
画面：店内环境快速横摇（3 秒内扫完）
口播：「店不大，工作日坐满了」

[9-13s] 菜品一出场
画面：第一道菜端上桌的瞬间，停留到定格
口播：「先来这个 XX，你看这个 XX（质感描述）」

[14-18s] 菜品一细节
画面：夹起/切开/流汁慢动作特写
口播：「一口下去 XX（口感），绝了」

[19-23s] 菜品二/三快切
画面：第二、第三道菜各 2-3 秒切换
口播：「这个 XX 也不能落下，XX 特别 XX」

[24-28s] 全桌俯拍
画面：全桌俯拍或斜上广角
口播：「这一桌才 XX 元，看看这分量」

[29-35s] 价格 + 转化
画面：价格贴纸特写 / 店门口招牌
口播：「团购链接挂好了，XX 区的朋友赶紧冲，手慢无」
```

## 菜品特写脚本模板（18 秒短视频）
```
[0-2s] 菜品出场
画面：厨师端菜上桌 / 从锅中盛出瞬间
BGM：卡点音乐切入

[3-6s] 微距质感
画面：菜品最诱人部位的微距特写（颜色/肌理）
字幕：菜名

[7-11s] 食欲动作
画面：拉丝/冒烟/浇汁/切开 慢动作（3-4 秒）
字幕：一句话卖点

[12-15s] 入口反应
画面：夹起 → 入口 → 咀嚼表情自然流露

[16-18s] 转化
口播/字幕：「在 XX，团购 XX 元」
画面：店名/地址字幕
```

## 后厨揭秘脚本模板
```
[0-3s] 钩子
口播：「你知道这碗 XX 为什么排队 2 小时吗？带你看后厨」
画面：店门口排队画面 / 锅里咕嘟沸腾

[4-8s] 食材亮相
画面：主要食材摆盘展示（新鲜感/产地/规格）
口播：「食材先看一眼，XX 都是当天的」

[9-13s] 关键工序一
画面：第一核心工序特写（手工/火候）
口播：「这一步 XX，很多地方省了，他们没有」
字幕：「XX 小时熬制」之类的关键信息

[14-18s] 关键工序二
画面：第二核心工序（现做/炭烤）
口播：「每天限量 XX 份，超了就没了」

[19-23s] 成品出锅瞬间
画面：成品从锅/蒸笼/烤炉出来的瞬间
口播：「你看这个状态——」

[24-28s] 装盘手法
画面：师傅装盘动作特写（体现专业度）
字幕：标注关键食材/工艺信息

[29-33s] 成品上桌
画面：成品端上桌，食客第一眼反应
口播：「这就是为什么好吃」

[34-38s] 转化
画面：店面外景 / 地址牌
口播：「地址在 XX，值得专门去一次」
```

## 拍摄要点
- 菜品特写用近景/微距，拍出拉丝、冒烟、流动等食欲感
- 保留环境音（炒菜声、咕嘟声）增加沉浸感
- 灯光偏暖，避免顶光把食物拍灰
- 入口反应真实自然，不要过度表演
- 竖屏拍摄，菜品放画面中心偏上

## 发布时间
饭前 11:00-12:00 / 17:00-18:00，夜宵 21:00-22:00，周四发布效果最佳

## 强钩子库（餐饮品类专属）

1. **藏了很久**：「在 [城市] 吃了 [年数] 年，这家小馆子我藏了 [时长]，今天告诉你」
2. **多次回头**：「人均 [价格] 元，这家我去了 [次数] 次还没吃够，[具体原因]」
3. **值不值问法**：「这碗 [菜名]，我专程绕路 [距离] 去买，值吗？你来判断」
4. **意外转折**：「服务员说这是招牌菜，我说没听说过，然后上了桌我沉默了」
5. **等待付出**：「排队 [时长]，刚进去第一口差点哭出来——[具体描述]」

**钩子使用原则**：一定要有具体数字（距离/次数/时间）；"等待"和"代价"能放大食物的价值感。

## 情绪语言包

**食物描述词**（告别"好吃""味道不错"）：

| 普通描述 | 情绪化替换 |
|---------|-----------|
| 味道很好 | 一口下去愣了三秒 / 吃到想哭 |
| 分量大 | 上桌的时候我倒吸一口气 |
| 汤很鲜 | 那个汤，我用了两个词：干净，然后上头 |
| 很香 | 端上来的时候旁边桌的人都看过来 |
| 性价比高 | 这一桌要是在别的地方吃，至少翻倍 |
| 值得推荐 | 这条视频发出去之前，我已经约了朋友再去一次 |
| 服务好 | 老板看到我拍照过来说：等下帮你拍一张 |

**探店口播节奏词**：
「你看」「来了来了」「等一下」「重点来了」「这一口」「不行了」「已经不想走了」

## 去广告感写法指引

| 硬广写法（❌避免） | 真实感替换（✅用这个） |
|-----------------|-------------------|
| 「食材新鲜，品质保证」 | 「厨师说每天早上五点进货，我进后厨看了，信了」 |
| 「环境优雅，适合聚餐」 | 「两张桌子之间不挤，能好好说话，这在 [城市] 很难得」 |
| 「人气爆棚，强烈推荐」 | 「工作日中午坐满了，我排了 [时长]，这不是广告，是真实情况」 |
| 「口味正宗，传承秘方」 | 「老板说配方三十年没变，我不知道真假，但这个味道让我想起 [记忆]」 |
| 「团购优惠，限时抢购」 | 「我帮你问了老板，他说这个价格只维持到 [时间]，怎么选你自己看」 |

**核心原则**：探店视频最怕"像是被请来的"。加入一个让观众相信你付了钱的细节（收据/排队/等待/自费买单的叙述），信任感立刻不同。
