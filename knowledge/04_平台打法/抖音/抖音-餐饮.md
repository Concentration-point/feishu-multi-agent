# 抖音脚本模板 — 餐饮/美食

## 适用客户
餐厅、奶茶店、烘焙店、火锅店、地方特色小吃、食品品牌

## 内容类型选择
| 类型 | 适用场景 | 时长 |
|------|---------|------|
| 探店口播 | 新店开业/团购推广 | 30-45 秒 |
| 菜品特写 | 招牌菜/新品上市 | 15-20 秒 |
| 后厨揭秘 | 品质背书/食材展示 | 20-40 秒 |
| 吃播反应 | 口味种草 | 15-30 秒 |
| 老板 IP | 品牌故事/行业揭秘 | 30-60 秒 |

## 探店口播脚本模板
```
[0-3s] 钩子
画面：推门进店 / 菜品大特写
口播：「在 XX 路发现一家 XX，人均才 XX 元！」

[3-25s] 菜品展示（最多 3 道，每道 5-7 秒）
画面：每道菜特写 → 夹起/切开 → 入口表情
口播：「先来这个 XX，你看这个 XX（质感/分量），
一口下去 XX（口感），绝了」

[25-35s] 套餐价值 + 转化
画面：全桌俯拍 / 价格贴纸
口播：「这一桌才 XX 元，团购链接挂好了，
XX 区的朋友赶紧冲」
```

## 菜品特写脚本模板（15-20 秒短视频）
```
[0-2s] 菜品出场
画面：厨师端菜上桌 / 从锅中盛出
BGM：卡点音乐

[2-12s] 食欲镜头组合
画面：微距拍质感 → 拉丝/冒烟/浇汁 → 夹起慢动作
字幕：菜名 + 一句话卖点

[12-18s] 吃 + 转化
画面：入口咀嚼表情
口播/字幕：「在 XX，团购 XX 元」
```

## 后厨揭秘脚本模板
```
[0-3s] 钩子
口播：「你知道这碗 XX 为什么排队 2 小时吗？带你看后厨」

[3-30s] 制作过程
画面：食材展示 → 关键工序特写（手打/现熬/炭烤）→ 成品出锅
口播：逐步讲解食材和工艺的讲究
字幕：标注关键信息（XX 小时熬制 / 每日限量 XX 份）

[30-38s] 成品 + 转化
画面：成品上桌 + 店面外景
口播：「这就是为什么好吃，地址在 XX」
```

## 拍摄要点
- 菜品特写用近景/微距，拍出拉丝、冒烟、流动等食欲感
- 保留环境音（炒菜声、咕嘟声）增加沉浸感
- 灯光偏暖，避免顶光把食物拍灰
- 入口反应真实自然，不要过度表演
- 竖屏拍摄，菜品放画面中心偏上
