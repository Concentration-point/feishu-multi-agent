# 抖音脚本模板 — 本地生活/服务业

## 适用客户
美发店、健身房、汽修店、宠物店、摄影工作室、月子中心、洗车店

## 内容类型选择
| 类型 | 适用场景 | 时长 |
|------|---------|------|
| 服务过程展示 | 技术/手艺展示 | 20-40 秒 |
| 客户变化对比 | 效果背书（发型/健身） | 15-30 秒 |
| 老板/员工 IP | 人设打造/日常 | 20-45 秒 |
| 福利活动 | 团购/到店引流 | 15-25 秒 |

## 服务过程展示脚本模板
```
[0-3s] 钩子
画面：客户到店，问题状态正面展示
口播：「这位姐姐说她 XX 很久了，今天找我解决」

[4-8s] 分析诊断
画面：师傅/技师近距查看问题部位
口播：「看了一下，主要问题是 XX，解法是 XX」

[9-13s] 服务过程·关键步骤一
画面：第一个核心操作手法特写
字幕：当前步骤名称
BGM：轻快/治愈音乐

[14-18s] 服务过程·关键步骤二
画面：第二个关键操作手法特写
口播：「这里注意 XX，很多地方会省这一步」

[19-23s] 中间阶段效果
画面：服务进行中的阶段性变化
口播：「做到这里你们看，已经 XX 了」

[24-28s] 最终成果展示
画面：最终效果正面 + 侧面各 2 秒
字幕：服务项目名称 + 价格

[29-33s] 客户真实反应
画面：客户对镜惊喜表情（真实捕捉，非摆拍）
口播：「看看效果，姐姐自己都不敢认了」

[34-40s] 转化
画面：门店外景 / 招牌
口播：「想要同款的私信我，地址在主页」
```

## 客户变化对比脚本模板（25 秒）
```
[0-2s] 变化前正面
画面：客户原始状态正面
字幕：「改造前」
BGM：卡点音乐

[3-5s] 变化前局部
画面：最明显的问题部位近景

[6-8s] 服务过程快闪一
画面：核心操作手法 2-3 秒
BGM：卡点

[9-11s] 服务过程快闪二
画面：另一操作镜头，节拍切换

[12-16s] 变化后正面
画面：成果正面展示（同机位）
字幕：「改造后」

[17-21s] 变化后细节
画面：改变最明显部位的近景特写
字幕：服务项目名称

[22-25s] 转化
字幕：「同城到店体验，私信预约」
画面：客户满意表情定格
```

## 福利活动脚本模板
```
[0-3s] 钩子（利益先行）
口播：「XX 区的朋友有福了，XX 只要 XX 元」
画面：服务/产品最直观画面

[4-8s] 内容展示
画面：服务过程实拍 / 产品实物展示
口播：「包含 XX，还有 XX」

[9-13s] 效果/品质背书
画面：成果展示 / 客户评价截图
口播：「做完效果是这样，真实拍的」

[14-18s] 限时限量
画面：价格数字大字展示
口播：「只限 XX 份，先到先得」

[19-22s] 转化
画面：团购链接二维码 / 引导截图
口播：「手慢无，链接挂好了」
```

## 拍摄要点
- 重点是「变化感」，前后对比要明显
- 操作手法用近景/特写体现专业度
- 客户反应真实自然，不要摆拍感
- 可用定机位延时拍摄整个服务过程

## 发布时间
工作日 11:00-12:00 / 17:00-19:00，周末 11:00-12:00 / 16:00-17:00

## 强钩子库（本地生活品类专属）

1. **替用户薅福利**：「同城的朋友，这个优惠我替你问来了，[具体优惠内容]，不看你亏大了」
2. **回购背书**：「找了 [时长] 的 [服务类型]，就这一位我回购了 [次数] 次，[一句话原因]」
3. **变化感**：「朋友说我换了个人，其实就是去了这家店 [做了什么]」
4. **情感时刻**：「[商家类型] 的老板今天说了一句话让我 [情绪反应]——[具体那句话]」
5. **归属感**：「在这个城市生活了 [年数] 年，这几家店不关门我不走——今天说一家」

**钩子使用原则**：本地生活最强的抓手是"同城人"标签 + 具体地名/街道，触达率和点击率都更高。

## 情绪语言包

**服务体验词**（替换"服务好""技术一流""性价比高"等通用词）：

| 通用描述 | 情绪化替换 |
|---------|-----------|
| 服务好 | 我进门的时候他记得我上次说什么，这让我有点受宠若惊 |
| 技术一流 | 做完之后我对着镜子站了 [时长]，没想到会是这个结果 |
| 价格实惠 | 买单的时候有点不确定数字，以为说错了，重新问了一遍 |
| 强烈推荐 | 我把这家推荐给了 [数量] 个朋友，到现在一个投诉都没有 |
| 下次还来 | 我出门的时候已经在想下次什么时候再来 |
| 效果满意 | 我走出来的时候停了一下，因为我在玻璃门里看了一眼自己 |

**本地生活口播节奏词**：
「同城的听好了」「就在 [地点]」「我来给你们报告一下」「说真的」「真的值得一去」「来过就知道」

## 去广告感写法指引

| 硬广写法（❌避免） | 真实感替换（✅用这个） |
|-----------------|-------------------|
| 「专业技师，技术精湛」 | 「他做这个 [年数] 年了，我坐下来跟他说我想要什么，他说'你说的那个不适合你脸型，换个方向试试'——这句话让我信任他了」 |
| 「环境舒适，设备先进」 | 「店里不大，但坐进去不觉得小，有种说不清楚的放松」 |
| 「优惠团购，名额有限」 | 「老板说这个价格做不了多久，我没办法帮你核实，但我自己这次是按这个价格付的，截图在这」 |
| 「好评众多，口碑极佳」 | 「我去的时候有 [人数] 个人在等，工作日下午 [时间点]——这个数字能说明一些问题」 |
| 「欢迎到店体验」 | 「如果你在 [区域] 附近，值得绕过去试一次，不满意我负责」 |

**核心原则**：本地生活内容的信任核心是"我真的去了"。一个真实细节（等候人数/具体价格/店员说的话）比十句推荐词更有说服力。
