# 项目执行记录

这里服务的是 orchestrator 主链路，记录项目是如何一步步跑出来的。

## 作用
给 copywriter、reviewer、project_manager 以及后续复盘使用。

## 典型内容
- Brief 解读结果
- 策略方案中间稿
- 样稿 / 全量内容
- reviewer 审核记录
- 返工记录
- 最终交付说明

## 对应当前项目里的阶段
- account_manager
- strategist
- copywriter
- reviewer
- project_manager

## 这层的价值
它不是长期标准答案，而是：
**这次项目真实执行过程的可回放材料。**

## 写入原则
- 允许自动写入
- 允许有过程性内容
- 允许保留中间状态
- 但应尽量避免与正式经验区混写
